# 第一章：基础概念入门

## 本章简介

本章是 MPI 学习的起点，带你理解并行计算的核心概念和 MPI 的定位。

## 知识点

- **并行计算解决的问题**：大规模计算任务（流体力学、深度学习、气候预测等）
- **MPI 定义**：Message Passing Interface，消息传递接口标准
- **分布式内存 vs 共享内存**：理解为什么需要消息传递
- **MPI 发展历史**：从 MPI-1 到 MPI-4 的演进
- **主流实现**：OpenMPI、MPICH、Intel MPI

## 核心概念

- **通信域 (Communicator)**：进程组的抽象，管理进程间通信
- **进程秩 (Rank)**：进程在通信域中的唯一标识
- **消息传递**：进程间交换数据的方式

## 学习目标

1. 理解为什么需要并行计算
2. 掌握 MPI 的定位和价值
3. 理解分布式内存架构下的通信模式
4. 了解 MPI 的发展历程和主流实现
