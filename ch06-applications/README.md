# 第六章：完整应用实例 - 二维 Jacobi 迭代

## 本章简介

本章通过一个完整的科学计算应用——二维 Jacobi 迭代求解泊松方程，来综合运用前面学到的 MPI 编程技能。

## 知识点

### 问题背景
- **泊松方程**：∇²u = f，是热传导、流体力学等问题的核心方程
- **Jacobi 迭代**：求解线性方程组的经典迭代方法
- **离散化**：将连续问题转化为离散网格上的计算

### 并行化思路：域分解
- **空间分解**：将网格划分为多个子区域，每个进程负责一部分
- **Ghost Cell（鬼边界）**：相邻进程边界处的数据副本
- **通信模式**：边界数据交换

### 实现要点
- **数据划分**：按行或按块划分网格
- **边界通信**：每轮迭代交换鬼边界数据
- **收敛判断**：全局归约判断所有进程是否收敛
- **性能优化**：减少通信次数、优化内存访问

## 核心概念

### Jacobi 迭代公式
```
u_new[i,j] = (u[i-1,j] + u[i+1,j] + u[i,j-1] + u[i,j+1] + h²*f[i,j]) / 4
```

### 2D 域分解
```
+---------+---------+---------+
|  P0     |   P1    |   P2    |
|         |         |         |
+---------+---------+---------+
|  P3     |   P4    |   P5    |
|         |         |         |
+---------+---------+---------+
```

### 鬼边界通信
```c
// 发送上边界到上方进程
MPI_Send(top_ghost, ..., neighbor_top, ...);
// 接收下方边界
MPI_Recv(bottom_ghost, ..., neighbor_bottom, ...);
```
